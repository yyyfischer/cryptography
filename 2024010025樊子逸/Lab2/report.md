# Lab2 多次填充攻击流密码 实验报告

## 方法说明
本次实验采用**多次一密攻击**（Many-Time Pad Attack）方法。由于流密码在重复使用同一密钥时，密文异或会消除密钥流，得到明文异或结果。利用空格字符（`0x20`）与英文字母异或会翻转大小写的规律，可以推断出各位置的明文字符，进而恢复密钥流，最终解密目标密文。

## 密钥流推断过程
1. **数据预处理**：将 11 段十六进制密文转换为字节串。
2. **逐字节分析**：对于每个字节位置，收集所有密文中该位置的字节。
3. **空格假设**：假设某个密文在该位置是空格（`0x20`），则密钥流字节 = 密文字节 XOR `0x20`。用此密钥流解密其他密文的同一位置，得到的明文若均为字母或空格，则该假设成立。
4. **选择最佳候选**：对每个位置，取所有假设中得分最高（解密出的字母/空格比例 >= 0.9）的作为最终密钥流字节。
5. **密钥流填充**：将恢复出的密钥流应用到所有密文，得到解密结果。
